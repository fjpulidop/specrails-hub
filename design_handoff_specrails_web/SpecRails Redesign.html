The Team

14 specialized agents.
One unified pipeline.

Three agents are core and always run — Architect, Developer, and Reviewer. The rest are optional specialists. The implement command dispatches the right Developer dynamically by task keywords, and the Reviewer sub-specializes into Frontend or Backend reviewers when installed. Compare all agents →
